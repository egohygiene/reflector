# Audits

This directory contains audit outputs, validation reports, and readiness assessments.

## Organization

- Active audit reports go directly in `audits/`
- Historical audit archives go in `audits/archive/`

## Naming Convention

- `YYYY-MM-DD-<topic>.md` for dated reports
- `<topic>-readiness.md` for readiness assessments
- `<topic>-audit.md` for recurring audits
