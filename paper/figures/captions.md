# Figure Caption Registry

Centralized registry of all figure captions, labels, and section placements for the Reflector paper.
Caption text here is the canonical source. Any update to a caption must be applied to the corresponding
`\caption{}` call in the LaTeX section file and kept in sync with this registry.

See [`manifest.md`](manifest.md) for full figure metadata and [`README.md`](README.md) for workflow guidance.

## Caption update workflow

1. Identify the figure by its canonical ID in [`manifest.md`](manifest.md).
2. Update the caption text in this file under the relevant heading.
3. Apply the identical caption text to the `\caption{}` call in the corresponding LaTeX section file.
4. Verify that the label (`\label{fig:...}`) in the LaTeX file matches the `label` field in [`manifest.md`](manifest.md).
5. Run `scripts/audit-publication-readiness.py` to confirm all figure integrity checks pass.
